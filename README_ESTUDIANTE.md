## Comandos Git utilizados

```bash
# Clonar el repositorio original
git clone https://github.com/DaneshCode/Portfolio-Website.git

# Verificar estado de los archivos
git status

# Agregar todos los cambios al área de staging
git add .

# Crear un commit con mensaje descriptivo
git commit -m "Personalización completa del portafolio profesional"

# Verificar el repositorio remoto actual
git remote -v

# Eliminar la referencia remota original
git remote remove origin

# Conectar con nuevo repositorio remoto propio
git remote add origin https://github.com/USUARIO/portafolio-profesional-web.git

# Renombrar la rama principal
git branch -M main

# Subir el proyecto a GitHub
git push -u origin main

# Descargar cambios del repositorio remoto
git pull origin main

# Ver ramas existentes
git branch
```

## Actividad opcional — GitHub Pages

Para publicar el sitio con GitHub Pages:

1. Ir a **Settings** del repositorio en GitHub
2. Scroll hasta **Pages**
3. En "Source", seleccionar rama **main** y carpeta **/ (root)**
4. Guardar y esperar unos minutos
5. La URL será: `https://USUARIO.github.io/portafolio-profesional-web/`
